# Сайт саженцев кустарников

## Описание проекта
Мобильный сайт для продажи саженцев кустарников.

## Технические характеристики
**Фреймворк**: React 18 с Babel для трансформации JSX
**Стили**: Tailwind CSS с кастомными CSS-переменными для цветовой схемы
**Иконки**: Lucide Static для минималистичных иконок
**Адаптивность**: Mobile-first подход, оптимизирован для экранов от 320px
**Обработка ошибок**: Реализована через `ErrorBoundary` и try-catch в каждом компоненте
**Изображения**: Используются оптимизированные ссылки на Unsplash с параметрами `auto=format&fit=crop&w=400&q=80`


## Структура сайта
1. **Header** - логотип и контактная кнопка
2. **Hero-секция** - главное предложение с промо-акцией
3. **Products** - карточки саженцев с ценами и характеристиками
4. **Features** - почему выбирают нас
5. **OrderForm** - простая форма для оформления покупки
6. **Footer** - контактная информация
7. **Navigation** - фиксированная панель навигации

## Цветовая схема
**Основной**: #2d5016 (тёмно-зелёный) — для цен, кнопок и акцентов
**Акцент**: #7cb518 (салатовый) — для иконок, меток "Популярный" и градиентов
**Акционная цена**: #ef4444 (красный, `bg-red-500`, `text-red-500`) — для меток акции и акционных цен
**Фон**: #f8fdf3 (светло-зелёный) — для секций Products и OrderForm
**Текст**: #1f2937 (тёмно-серый) — основной текст, #6b7280 (светло-серый) — второстепенный текст
**Градиенты**: От #7cb518 (салатовый) к #2d5016 (тёмно-зелёный) — для фонов иконок и кнопок
